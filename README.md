# Backend - Tienda Fut.Players

Este es el servidor API REST para la tienda de camisetas de fútbol, desarrollado con Java Spring Boot y MySQL.

Requisitos Previos
1. Tener instalado Java JDK 17 o 21.
2. Tener instalado XAMPP (MySQL) corriendo en el puerto 3306 (o 3307 si lo configuró así).
3. Tener Maven instalado (o usar el wrapper incluido).


Instrucciones de Ejecución
Dar Run en BaquentAplicacion.java


El servidor iniciará en: http://localhost:8080
